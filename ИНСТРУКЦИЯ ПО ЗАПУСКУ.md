# 🤖 Инструкция по запуску бота «Магия цифр»

## Что умеет бот
- В чате «Магия цифр» видит хэштеги
- `#ЗапросГуру` или `#Астрологикус` → отвечает Великий Гуру Астрологикус 🔮
- `#ЗапросАрви` или `#Арви` → отвечает Детектив Арви 🕵️
- Отвечает коротко, с крючком, нативно предлагает консультацию

---

## ШАГ 1 — Создать бота в Телеграм (5 минут)

1. Откройте Телеграм → найдите **@BotFather**
2. Напишите `/newbot`
3. Придумайте имя: например **Магия цифр Ассистент**
4. Придумайте username (латиницей, заканчивается на bot): например `magiya_chisel_bot`
5. BotFather пришлёт **токен** — длинный код вида `7xxxxxxxxx:AAF...`
6. Скопируйте этот токен — он нужен в шаге 3

---

## ШАГ 2 — Получить ключ OpenAI (10 минут)

1. Зайдите на **platform.openai.com**
2. Зарегистрируйтесь (нужна карта — спишут $5 для старта)
3. Перейдите: **API Keys → Create new secret key**
4. Скопируйте ключ вида `sk-...`

---

## ШАГ 3 — Запустить на бесплатном сервере Railway (15 минут)

1. Зайдите на **railway.app** → Sign in with GitHub
2. Нажмите **New Project → Deploy from GitHub repo**
3. Загрузите папку `magiya-chisel-bot` на GitHub (я помогу)
4. В Railway добавьте переменные окружения:
   - `TELEGRAM_TOKEN` = ваш токен от BotFather
   - `OPENAI_API_KEY` = ваш ключ от OpenAI
5. Railway запустит бота автоматически

---

## ШАГ 4 — Добавить бота в чат (2 минуты)

1. Зайдите в чат **«Магия цифр»**
2. Настройки чата → Добавить участника
3. Найдите вашего бота по username
4. Сделайте его **администратором** (иначе не сможет читать сообщения)

---

## КАК ПОЛЬЗОВАТЬСЯ

Участник пишет в чат:
```
#ЗапросГуру Мне 35 лет, родилась 15.07.1989. Почему никак не складывается карьера?
```

Бот автоматически отвечает от имени Астрологикуса — коротко, тепло, с предложением записаться.

```
#ЗапросАрви Хочу создать вирусное видео для своего канала. С чего начать?
```

Бот отвечает от имени Арви — с вердиктом, уликами, практическим советом.
